$ cat why_bart.md

BART is an encoder-decoder seq2seq transformer. Clause extraction was framed as conditional generation: input a tagged clause, output structured JSON fields.

ApproachMechanismLimitation
BERT QA (span extraction)Finds answer span in source textFails when answer requires inference — not a literal span
BART seq2seqGenerates structured output from encoderHandles paraphrase, inference, multi-sentence clauses
